Settings Menu

Opened from Tools -> Settings on the menubar, this dialog allows the user to permanently modify or reset several configuration variables of the application. Settings are stored in a text file found at .taskstalker/config within the user's home directory, and are loaded into the program at launch time.



1) Input: Process Table Update Interval

This input box sets how regularly the process table in the main window is updated, in seconds. Minimum 1 second, maximum 60 seconds.

2) Input: Sub-Process Table Update Interval

Similarly to above, this sets how often the process information is updated in the Visualisation and Simulation dialogs. Minimum 1 second, maximum 60 seconds.

3) Menu: Process Table Sorting

Selects the column of the main window process table by which to sort process data.

4) Menu: Process Table Sort Order

Set the previously described sorting to be performed in ascending or descending order.

5) Cancel Button

Exit the settings menu, making no changes to the configuration file or running configuration.

6) Reset All Button

Exit the settings menu, reverting all running configuration variables to their default values, and updating the configuration file.

7) Save Button

Exit the settings menu, modifying the running configuration to match selections made in the settings menu then updating the configuration file.
