Simulation Window

Opened from Tools -> Simulate on the menubar, this dialog runs a program on the system and monitors its state, displaying via a visualisation dialog. The program must be provided by the user.



1) Program Path Input

This text field allows the user to enter a path to a program to execute using the simulator module. The path may be relative to this application's location, such as "../program", or absolute, such as "/usr/bin/program".

2) Cancel Button

Clicking this button returns the user to the main window.

3) Use Default Button

Ignoring any input in the text field above, this button launches the simulator, monitoring the default program's execution.

4) Continue Button

Launches the simulator, with the program specified in the above text field. If an invalid path is specified, or for any reason the program cannot be executed, the user is informed of the error and launching of the simulation module is cancelled.
